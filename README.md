# scala-web-gradle-spring-hello-world

## Description
A minimalist build for springframework
on scala.

## Tech stack
- gradle
- scala

## Docker stack
- gradle:jre11

## To run
`sudo ./install.sh -u`
- Available at http://localhost

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
